# Memrise [![Build Status](http://travis-ci.org/doriath/memrise.png)](http://travis-ci.org/doriath/memrise)

Memrise (www.memrise.com) API wrapper.

## Installation

Add this line to your application's Gemfile:

    gem 'memrise'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install memrise
